# MoodAnalyzer

## Repository Name
mood-analyzer

## Project Name
MoodAnalyzer

## Project Description
MoodAnalyzer is a web application that detects the mood or emotion of a text input using Natural Language Processing (NLP) techniques. Users can enter any text, and the app will analyze it to determine whether the mood is **Positive/Happy**, **Negative/Sad**, or **Neutral**.

The application is built with **Django** for the backend and uses **HTML, CSS, and JavaScript** for the frontend. It is designed to be simple, user-friendly, and informative.

---

## Features
- Detects mood from user input text.
- Simple and clean user interface.
- About page with team member details including names, roles, and email IDs.
- Error handling for blank or invalid inputs.

---

## Technologies Used
- **Backend:** Python, Django, TextBlob (for sentiment analysis)  
- **Frontend:** HTML5, CSS3, JavaScript  
- **Version Control:** Git, GitHub
